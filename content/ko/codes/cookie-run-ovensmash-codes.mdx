## 쿠키런: 오븐스매시 쿠폰 등록 방법

게임 내에 등록 메뉴가 있는 일부 모바일 게임과 달리, 이 게임은 데브시스터즈의 전통적인 형식을 따릅니다. 계정을 연결하고 아이템을 받으려면 외부 웹 포털을 사용해야 합니다. 보상이 게임 내 우편함으로 정확히 전달되도록 다음 단계를 정확히 따르세요.

### 1단계: 플레이어 ID 확인
웹사이트로 이동하기 전에 고유한 DevPlay 계정 ID가 필요합니다. 이는 시스템이 어떤 계정에 크리스탈을 지급해야 하는지 식별하는 번호입니다.

1. 게임을 실행하고 메인 로비로 들어갑니다.
2. 화면 우측 상단에 있는 **세 줄 모양의 아이콘**(메뉴)을 탭합니다.
3. **설정** 톱니바퀴 아이콘을 선택합니다.
4. 왼쪽의 **계정** 탭으로 이동합니다.
5. **플레이어 ID**(또는 DevPlay ID)를 찾아 옆에 있는 복사 버튼을 탭합니다.

### 2단계: 쿠폰 등록 페이지 접속
ID를 확인했다면, 공식 [데브시스터즈 쿠폰 페이지](https://www.devsisters.com/en) 또는 오븐스매시 전용 등록 사이트를 방문해야 합니다.

| 작업 | 설명 |
| :--- | :--- |
| **DevPlay 계정 입력** | 게임 설정에서 복사한 플레이어 ID를 붙여넣습니다. |
| **쿠폰 코드 입력** | 활성 코드(예: **WELCOME2OVENSMASH**)를 입력합니다. |
| **보상 받기** | 버튼을 클릭하여 프로세스를 완료합니다. |

### 3단계: 보상 수령
웹사이트에서 등록 성공이 확인된 후, 아이템이 인벤토리에 즉시 나타나지는 않습니다. 게임으로 돌아가 **우편함**을 확인해야 합니다. 보상이 즉시 보이지 않는다면 앱을 재시작하여 서버 연결을 새로고침해 보세요.

**쿠키런 오븐스매시 쿠폰 코드**를 사용할 때 오류가 발생한다면 보통 몇 가지 일반적인 요인 때문입니다. 고객 지원에 문의하기 전에 다음 사항을 확인하세요.

*   **잘못된 플레이어 ID:** 표시 이름이 아닌 계정 탭의 DevPlay ID를 사용하고 있는지 확인하세요. 표시 이름은 고유하지 않으며 쿠폰 등록에 사용할 수 없습니다.
*   **지역 제한:** 때때로 특정 코드는 특정 지역 서버(소프트 런칭 지역 등) 전용으로 제한될 수 있습니다. 사용 중인 코드가 본인의 글로벌 또는 로컬 버전에 적합한지 확인하세요.
