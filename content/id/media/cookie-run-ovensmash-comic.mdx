### Cara Membuat Komik Pertama Anda

1.  **Pilih Kanvas Anda:** Pilih antara tata letak panel standar atau Flipbook Mode panel tunggal untuk animasi.
2.  **Impor Sprite:** Gunakan tab "User-Submitted Sprites" atau "Studio Crossover" untuk menemukan karakter seperti String Cheese Cookie atau penghuni baru Meat District.
3.  **Atur Posisi:** Gunakan "Sprite Panel Lock" untuk menjaga karakter tetap di dalam batas, atau nonaktifkan untuk efek "menembus dinding keempat".
4.  **Tambahkan Dialog:** Tempatkan gelembung ucapan dan sesuaikan warna teks agar sesuai dengan estetika karakter (misalnya, emas untuk Brie Cheese Cookie).
5.  **Ekspor dan Bagikan:** Simpan data Anda secara lokal atau ekspor sebagai gambar resolusi tinggi untuk dibagikan kepada komunitas.
