## วัตถุประสงค์ในแผนที่และกลยุทธ์ทีม

เกมเพลย์หลักจะหมุนรอบการต่อสู้ระหว่างทีมสีแดงและทีมสีน้ำเงิน แต่ละทีมมีปราสาทที่ต้องปกป้องในขณะที่ต้องโจมตีป้อมปราการของฝ่ายตรงข้ามไปพร้อมๆ กัน

### กลไกการบุก (The Push Mechanic)
เมื่อคุณกำจัดศัตรูและเคลียร์สิ่งกีดขวาง เกจ "Push" ของคุณจะเพิ่มขึ้น การบุกที่แข็งแกร่งจะช่วยให้ทีมของคุณสร้างความเสียหายมหาศาลแก่ปราสาทของศัตรู เมื่อผู้ประกาศตะโกนว่า "ปราสาทของทีมสีแดงเหลือพลังชีวิตครึ่งเดียว" นั่นคือสัญญาณให้คุณทุ่มสุดตัวในการโจมตีหรือถอยกลับมาตั้งรับอย่างสุดชีวิต

### Overtime และ Sudden Death
หากการแข่งขันยังไม่รู้ผลภายในเวลามาตรฐาน เกมจะเข้าสู่ช่วง "Overtime" ในช่วงนี้ เวลาในการเกิดใหม่จะเพิ่มขึ้น และพลังของคุกกี้ทุกตัวจะถูกขยายให้สูงขึ้น นี่เป็นช่วงเวลาที่พบบ่อยที่สุดในการทำ "Penta Kills" เนื่องจากแผนที่เล็กลงและความกดดันสูงขึ้น

> **คำเตือน:** ในช่วง Overtime การเสียเพื่อนร่วมทีมแม้เพียงคนเดียวอาจนำไปสู่ความพ่ายแพ้ในทันที จงเกาะกลุ่มกันไว้และระดมยิงไปที่เป้าหมายเดียวเพื่อชิงความได้เปรียบ

ตอบ: ใช่ Cookie Run OvenSmash ถูกออกแบบมาเพื่อการเล่นข้ามแพลตฟอร์ม (Cross-platform) ทั้งบน iOS, Android และ PC ทำให้ผู้เล่นสามารถแข่งขันกันได้ไม่ว่าจะใช้อุปกรณ์ใดก็ตาม

**ถาม: วิธีที่ดีที่สุดในการใช้พาวเวอร์บิสกิตคืออะไร?**

ตอบ: คุณควรให้ความสำคัญกับการเพิ่มเลเวลทักษะความเสียหายหลักของคุกกี้ก่อน อย่างไรก็ตาม หากคุณพบว่าตัวเองตายบ่อย การลงทุนในบิสกิตป้องกันสามารถช่วยให้คุณอยู่ในการต่อสู้ได้นานขึ้น

**ถาม: ฉันจะได้รับตำแหน่ง Grand Oven Crown ได้อย่างไร?**

ตอบ: ตำแหน่ง Grand Oven Crown จะมอบให้กับ MVP ของทีมที่ชนะในโหมด Crown Cup ซึ่งต้องอาศัยการผสมผสานระหว่างการมีส่วนร่วมในการฆ่าที่สูง การทำดาเมจต่อวัตถุประสงค์ และการเก็บเหรียญ

**ถาม: ฉันสามารถหาตัวอย่างล่าสุดของ Cookie Run OvenSmash YouTube ได้ที่ไหน?**

ตอบ: ช่อง YouTube อย่างเป็นทางการของ Cookie Run จะอัปโหลดตัวอย่าง การแนะนำตัวละคร และการอัปเดตจากผู้พัฒนาอยู่เป็นประจำ การค้นหา **cookie run ovensmash youtube** จะช่วยให้คุณพบกับคอนเทนต์และคู่มือที่สร้างโดยชุมชนที่หลากหลายเช่นกัน
