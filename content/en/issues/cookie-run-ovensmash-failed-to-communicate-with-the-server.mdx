### How to Change DNS on Mobile Devices

1.  **Android:** Go to Settings > Network & Internet > Private DNS. Select "Private DNS provider hostname" and enter `dns.google`.
2.  **iOS:** Go to Settings > Wi-Fi. Tap the 'i' next to your network, select "Configure DNS," switch to "Manual," and add `8.8.8.8` and `8.8.4.4`.

## Clearing App Cache and Data

Over time, the temporary files stored by the game can become fragmented or corrupted, especially after a large patch or seasonal update. Clearing the cache is a non-destructive way to force the game to re-download essential configuration files from the server.

1.  Open your device **Settings**.
2.  Navigate to **Apps** or **Applications Manager**.
3.  Locate **Cookie Run: OvenSmash** in the list.
4.  Select **Storage**.
5.  Tap **Clear Cache** (Do not tap "Clear Data" unless you are prepared to re-download the entire game and log in again).

> **Tip:** If you are playing on a PC via Steam, use the "Verify Integrity of Game Files" tool found in the game's properties menu to repair any corrupted installation data.

## FAQ

**Q: Why does the communication error only happen during matches?**

A: This usually indicates a "Network Unstable" condition. While your initial login might work, the real-time data requirements of a match are higher. If your ping spikes or your jitter is too high, the server will drop the connection to maintain game integrity for other players.

**Q: Will I lose my rank if the game fails to communicate with the server?**

A: In most cases, if the server detects a legitimate disconnection, it may trigger "loss prevention" mechanics. However, if the disconnection is frequent, the system might flag it as "suspicious gameplay," so it is best to fix the issue before playing more ranked matches.

**Q: Does using a VPN help with the cookie run ovensmash failed to communicate with the server error?**

A: It depends. If your ISP is poorly routing data to the game's specific region, a VPN can provide a more direct path. However, a poor-quality VPN will increase your latency (ping) and may actually make the communication errors more frequent.

**Q: How often does DevSisters perform server maintenance in 2026?**

A: Maintenance typically occurs once every two weeks, usually on Tuesdays or Wednesdays, to coincide with new character releases or balance patches. Always check the in-game "Notices" tab for exact timings to avoid disconnections.
