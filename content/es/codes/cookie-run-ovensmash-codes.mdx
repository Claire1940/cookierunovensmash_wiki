### Paso 1: Localiza tu ID de jugador
Antes de dirigirte al sitio web, necesitas tu ID de cuenta DevPlay único. Este es el identificador que le indica al sistema qué cuenta debe recibir los Cristales.

1. Inicia el juego y entra al vestíbulo principal.

3. Selecciona el icono de engranaje de **Ajustes**.
4. Navega hasta la pestaña **Cuenta** en el lado izquierdo.
5. Busca el **ID de jugador** (o ID de DevPlay) y toca el botón de copiar que aparece al lado.

### Paso 2: Accede al portal de canje
Una vez que tengas tu ID, debes visitar la [Página de cupones oficial de Devsisters](https://www.devsisters.com/en) o el sitio de canje específico de OvenSmash.

### Paso 3: Recoge tu botín
Después de que el sitio web confirme un canje exitoso, los artículos no aparecerán instantáneamente en tu inventario. Debes regresar al juego y revisar tu **Buzón**. Si las recompensas no aparecen de inmediato, intenta reiniciar la aplicación para refrescar la conexión con el servidor.

## Solución de problemas comunes con los códigos

Si encuentras un error al intentar usar **códigos de cookie run ovensmash**, generalmente se debe a uno de estos factores comunes. Antes de contactar al soporte, verifica lo siguiente:

*   **Códigos caducados:** La mayoría de los códigos promocionales solo duran 30 días o hasta el final de un mes de evento específico. Si un código figura como caducado en los foros de la comunidad, ya no funcionará.
*   **ID de jugador incorrecto:** Asegúrate de estar usando el ID de DevPlay de la pestaña Cuenta, no tu nombre de pantalla. El nombre de pantalla no es único y no se puede usar para el canje.
*   **Restricciones regionales:** Ocasionalmente, ciertos códigos están bloqueados para servidores regionales específicos (como regiones de Soft Launch). Asegúrate de que el código que estás usando esté destinado a tu versión global o local.
*   **Uso único:** Estos cupones están limitados a un solo canje por cuenta. Intentar usar el mismo código dos veces resultará en un error de "Ya canjeado".
